手順

Apacheのインストール

1.サーバへSSH接続し、以下のコマンドでルートユーザに昇格する。

sudo su

2.Apacheをインストールする。

yum install httpd

3.インストールされているかどうかを確認する。

yum list installed | grep httpd

以下のように出力されれればインストールできてます。
httpd.x86_64                             2.2.34-1.15.amzn1            @amzn-main
httpd-tools.x86_64                       2.2.34-1.15.amzn1            @amzn-main

Apacheの自動起動設定

1.Red Hat 7系からはsystemctlコマンドを使用してを自動起動に設定する。6系までは機能していたchkconfigコマンは7では使用できない。

#httpdの自動起動設定を確認
systemctl is-enabled httpd

disabled #自動起動設定：OFF

#httpdの自動起動を設定
systemctl enabled httpd

Created symlink from /etc/systemd/system/multi-user.target.wants/httpd.service to /usr/lib/systemd/system/httpd.service.

#httpdの自動起動設定を確認
systemctl is-enabled httpd

enabled #自動起動設定：ON

登録したDNSの内容を確認する

1.ムームードメインとAWS route53で取得ドメインを登録する。

2.反映までしばらく待つ。

3.名前解決できるかを確認する。

ApacheにFQDNを紐づける

1./etc/httpd/conf/ディレクトリに移動。

cd /etc/httpd/conf/
pwd

2./etc/httpd/conf/httpd.confファイルの中身をみる。長いのでviewコマンドで見てみる。

view httpd.conf

#viコマンドと同様に「:q」で終了できます。

3.バックアップを取得する。

#保存先ディレクトリを作成
mkdir reg-ConfigBackup

#保存先ディレクトリへバックアップを作成
cp -a httpd.conf ./reg-ConfigBackup/htppd.conf_yyyymmdd

#保存先ディレクトリにバックアップがとれていることを確認
ls -lh ./reg-ConfigBackup

4.ファイルを編集する。

#Apachの設定ファイルを確認してみよう
vi httpd.conf

#ファイルが開けましたらコマンドモード上で「/ServerName」と入力し、「Enter」キーを押下してください。

#以下の記述に飛べばおk。とばなかったら「N」キーをいくつか押下してみましょう。
##ServerName www.example.com:80

#「a」キーを押下し、インサートモードへ移行後、「www.example.com」の部分を「XXXX.teruya.okinawa」に変更してください。「xxxx」はなんでもおk

5.サービスを再起動する。

#Apacheのサービスを再起動
systemctl restart httpd

#Apacheのサービスを状態を確認
#起動していること
systemctl status httpd
