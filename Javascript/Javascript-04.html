3.スライダーはフィルムのようにコンテンツを直線上に並べておく必要がある。 上記のリンクではいくつかのdivがカラム落ちしているはず。（多分。。。）

ですので、それを防ぐよう努力します。以下をcssに追記する。

.slider .slideSet {
  width: 2500px;
}

追記後、保存して以下のように表示されていればうまくいってる。

リンクはこちら

4.次に、直線状に並べたコンテンツに対し、見えるコンテンツを限定する。 以下をcssに追記する。

.slider {
  width: 500px;
  height: 200px;
  overflow: hidden;
}

追記後、保存して以下のように表示されていればうまくいってる。
