css追記 --- h1、h2、clearfixのcss ---

1.css部分に以下を追記する。

追記部分

        h1 {
            font-size: 200%;
            text-align: center;
            padding: 20px;
        }
        h1 a {
            color: #444;
            text-decoration: none;
        }
        h1 a:hover {
            color: #888;
            text-decoration: underline;
        }
        h2 {
            font-size: 200%;
            margin-bottom: 0.5em;
        }
        p {
            margin-bottom: 1em;
        }
        .clearfix:before,
        .clearfix:after {
            content:"";
            display:table;
        }
        .clearfix:after {
            clear:both;
        }
        .clearfix {
            zoom:1;
        }
