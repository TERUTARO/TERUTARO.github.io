MySQLのインストール

MySQLの利用できるパッケージの種類

ここで、MySQLのパッケージについて紹介しましょう。

MySQLには、以下の4タイプがあります。

No 種類
1 MySQL Standard Edition
2 MySQL Enterprise Edition
3 MySQL Cluster Carrier Grade Edition
4 MySQL Community Server

名前でだいたい検討がつきますが、No.1～3の方は、有償であり、サポートおよび様々なツールが付随します。 No.4は無償ダウンロードが可能ですので、今回は、そちらを利用します。

ちなみに以下のページがMySQL Editionに関する詳細ページとなります。

https://www.mysql.com/jp/products/

パッケージ内容の確認

パッケージの内容を覗いてみましょう。 以下のコマンドを実行してみてください。

[コマンド]

 yum info mysql-community-server

結果がえられましたか？ 「バージョン」項目が8系のままだとさっきのリポジトリの有効／無効がうまくいってないかもしれませんね。

MySQLインストール

では、無償版をインストールしてみましょう！ 以下のコマンドを実行して、MySQLを実行してみましょう。

[コマンド]

yum -y install mysql-community-server

完了したかな？ 完了したら、コマンドを打って、バージョン情報を確認してみましょう。

[コマンド]

mysql —version

以下のように出力されればOKです！

[実行結果]

mysqld  Ver 5.7.22 for Linux on x86_64 (MySQL Community Server (GPL))

5.7系がちゃんとインストールできていれば、今回インストールしたいバージョンのMySQLがちゃんとインストールできたことになります。

サーバをシャットダウンした際に、サービス（MySQL）が自動的に

実行されるととても楽ですよね！ ということでサービスの自動起動をONにしましょう。

chkconfig mysql on ※chkconfig の「on」オプションでは、ランレベルとしては全て有効状態です。
