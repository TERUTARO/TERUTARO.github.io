MySQLを起動してみる¶

1.MySQLのサービスを起動する。

#MySQLのサービス状態を確認
systemctl status mysqld

#MySQLのサービスを起動
systemctl start mysqld

#MySQLのサービス状態を確認
systemctl status mysqld

MySQLの自動起動設定¶

1.systemctlコマンドにてmysqldを自動起動設定にする。


初期パスワードの変更¶

1.mysqlにログインします。

#mysqlにrootでログインします。
#この時にパスワードを聞かれますが、なにも入力せずEnterしてください。
mysql -u root -p

#パスワードを設定
 set password for root@localhost=password('パスワード');

#一度ログアウト
quit

2.再ログインしてパスワードが設定されていることを確認

#ログイン
mysql -u root -p

データベースの作成¶

1.MySQLへログイン

#mysqlへログイン
mysql -u root -p

2.データベースを作成しましょう。 今回は下表のようなデータベースを作成します。

DB名:test

mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+
4 rows in set (0.01 sec)

mysql>

では新たにデータベースを作成しましょう。

#"test"という名前のデータベースを作成
create database test;

mysql> create database test;
Query OK, 1 row affected (0.00 sec)

show databases;

mysql> show databases;
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
| test               |
+--------------------+
5 rows in set (0.01 sec)

次に、testデータベースに接続してみましょう。

#"test"データベースに接続
use test;

#Changeとかでる。
Database changed

これで接続できました。 では次にカラムを追加しましょう。

＃"user"という名称のテーブルを作成
create table user;

#カラムを一気に設定しましょう。
create table user (id int auto_increment primary key,username varchar(255),dibision varchar(255),age char(200));

#できたかを確認
#"user"という名称のテーブルができているように見えますね。
mysql> show tables;
+----------------+
| Tables_in_test |
+----------------+
| user           |
+----------------+
1 row in set (0.00 sec)



レコード(データ)の追加¶

1.次のコマンドでレコードを追加していきましょう。

#"inert"コマンドで追加していきます。
#渡辺さんを追加
insert into user(username,dibision,age)
values('watanabe','infra','30');

#森田さんを追加
insert into user(username,dibision,age)
values('morita','infra','23');

#川島さんを追加
insert into user(username,dibision,age)
values('kawashima','kikaku-unyo','29');

#照屋さんを追加
insert into user(username,dibision,age)
values('teruya','kikaku-unyo','26');


#追加レコード内容を確認
select *from user;

2.レコード内容にミスがあった場合は次の様にして修正しましょう。

#updateコマンドで内容を更新
update user set username = watanabe where id = 1;

#変更内容を確認
select *from user;
